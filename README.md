# crawler_imagens

## How to install

### Clone the repo

`git clone https://github.com/acszec/crawler_images.git`

### Create a virtualenv

`mkvirtualenv crawler-images-env`

### Set virtual env project

`cd crawler_images && setvirtualenvproject`

### Install dependecies

`make install`

### Start getting all link

`make start`

### Download all images

`make download`
